User Manual

Complete guide to using the LA-MALIVA VISTA HOTEL Management System

Getting Started Dashboard Overview Managing Reservations Check-in & Check-out Billing & Invoices Admin Tools

1. Getting Started

Welcome to the La-Maliva Vista Hotel PMS. This system allows you to manage rooms, guests, and bookings efficiently.

  • Login: Access the system at /login using your username and password.
  • Roles:
    • Admin Full access to all settings, users, and financial data.
    • Staff Access to reservations, guests, and daily operations.

2. Dashboard Overview

The dashboard is your command center. It provides a real-time snapshot of the hotel's status.

Key Metrics

View total rooms, occupied rooms, and current occupancy percentage instantly.

Today's Activity

See a list of guests arriving today (Check-ins) and guests departing today (Check-outs).

3. Managing Reservations

You can create bookings via the "New Reservation" button or the Calendar view.

  1. Click New Booking in the toolbar.
  2. Enter guest details (Name, Phone, Email).
  3. Select a Room Type (Standard, Deluxe, Suite, Family).
  4. Choose Check-in and Check-out dates/times.
  5. Click Create Reservation.
Tip: Use the Calendar view to drag and drop bookings or see gaps in the schedule.

4. Check-in & Check-out

Managing guest flow is simple:

Check-in Process

Go to Today's Arrivals or the Calendar. Click the green button. The room status will change to "Occupied".

Check-out Process

Go to Departures. Click . The room will become "Available" again.

5. Billing & Invoices

Generate professional invoices for any reservation.

  • Navigate to a reservation details page.
  • Click the Invoice button.
  • Review the details (Room charges + duration).
  • Use the Print button for a physical copy or Download PDF for a digital file.

6. Admin Tools (Admins Only)

Administrators have exclusive access to the Tools menu.

  • User Management: Add new staff accounts or remove old ones.
  • Settings: Update hotel name, address, and tax rates.
  • Backup/Restore: Download a copy of the database (`lamaliva.db`) to keep your data safe.
  • Export Data: Download guest lists as CSV files for Excel.
